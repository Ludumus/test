# WBA Analytics - Техническое задание - view_item_in_list

### <span style="color:darkgreen;">Интро</span>

На сайте и в приложениях iOS/Android пользователь может просмотреть сниппеты товаров в выдачах товаров, сетках рекомендаций, каруселях и т.п.

### <span style="color:darkgreen;">Дополнительная информация</span>

* Если у карусели товаров есть функционал, при котором есть возможность нажать на кнопку “Все товары” и попасть в выдачу со всеми товарами этой карусели, то товары на экране за кнопкой “Все товары” также должны быть размечены событием
* Сниппеты товаров в разделах “Корзина” и “Избранное” размечать не следует. Это относится только к добавленным товарам в корзину/избранное. Если присутствуют рекомендованные товары и т.п., то их следует разметить.

### <span style="color:darkgreen;">Ожидаемый результат</span>

Событие `view_item_in_list` размечено вместе с атрибутами объектов **items** (Mobile SDK)/ **products** (Web SDK) согласно таблице ниже

### <span style="color:darkgreen;">Техническое задание</span>

**Событие:**
`view_item_in_list` - просмотр сниппета товара

**Триггеры:**
Пользователь увидел 50% сниппета товара

**Кастомные параметры:**
Нет
